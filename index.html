Intro.js

Better introductions for websites and features with a step-by-step guide for your projects.

Show me how
Tweet

Easy to Use

Just include the js and css files and add data-intro-step and data-intro-text to your code.

Fast & Small

4 KB JavaScript and 2.5 KB CSS (gzip). That's all.

Free & Open-Source

Free and open-source (including commercial use). Published under MIT license.

Keyboard + Mouse navigation

Ability to navigate with the mouse, or the keyboard - ←, →, ENTER and ESC to exit.

Browser Compatibllity

Better compatibility is on the roadmap, but currently it should work in recent versions of Firefox, Chrome and IE8 also.
